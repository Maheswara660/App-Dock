> [!IMPORTANT]
> **Privacy Manifesto**: App Dock is built on a strict privacy-first, local-only philosophy. We believe your workspace and data belong entirely to you. No telemetry, no accounts, no cloud sync—just your apps, your way.

---

## ✨ Features

> [!NOTE]
> App Dock bridges the gap between web and native, providing a seamless workspace experience across platforms.

- 🛡️ **Total Privacy**: Everything stays on your machine. No tracking, no data collection.
- 🚀 **Modern UI**: Sleek, responsive design powered by Compose Multiplatform.
- 📦 **Native Excellence**: Transform websites into standalone windows with dedicated icons.
- 🛠️ **True Multi-Platform**: Consistent experience across Linux, Windows, and Android.

## 🚀 Installation Instructions

### 🪟 Windows Installation
1. Navigate to the [Latest Releases](https://github.com/Maheswara660/App-Dock/releases) page.
2. Download the `App-Dock-Setup.exe` (Standalone Installer) or `App-Dock.msi` (Windows Installer).
3. Run the downloaded file and follow the on-screen prompts.
4. Launch **App Dock** from your desktop or Start menu.

---

### 🐧 Linux Installation

#### **Arch Linux (Manual Build from Source)**
We provide a `PKGBUILD` for Arch-based distributions (Manjaro, EndeavourOS, etc.) to ensure the best performance.
```bash
# 1. Clone the repository
git clone https://github.com/Maheswara660/App-Dock.git
cd App-Dock

# 2. Build and install automatically (requires base-devel and java-environment>=17)
makepkg -si
```

#### **AppImage (Universal Linux)**
1. Download the `App-Dock-*.AppImage` file from the releases.
2. Right-click the file -> Properties -> Permissions -> **Allow executing file as program**.
   *Or via terminal:* `chmod +x App-Dock-*.AppImage`
3. Double-click the file to run.

#### **Debian / Ubuntu / Kali / Mint**
1. Download the `.deb` package.
2. Run: `sudo dpkg -i App-Dock-*.deb`
3. Fix dependencies if needed: `sudo apt install -f`

#### **Fedora / RHEL / CentOS / OpenSUSE**
1. Download the `.rpm` package.
2. Run: `sudo rpm -i App-Dock-*.rpm`

---

### 🤖 Android Installation
1. Download the `App-Dock-*.apk` from the releases.
2. Open the file on your Android device.
3. If prompted, go to **Settings** and enable **"Install from Unknown Sources"**.
4. Complete the installation and open the app.
